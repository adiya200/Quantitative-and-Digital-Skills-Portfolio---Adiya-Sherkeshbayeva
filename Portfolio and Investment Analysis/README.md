Project 1: Portfolio and Investment Analysis
Overview
This project focuses on the quantitative analysis of stock market data to evaluate risk and return profiles. I analyzed NVIDIA (NVDA) and AMD performance relative to the S&P 500 benchmark using a year of weekly historical data.
Quantitative Skills Demonstrated
Regression Analysis: Built OLS regression models to determine stock Beta and R-Squared values.
Descriptive Statistics: Calculated Mean, Variance, Standard Deviation, and Coefficient of Variation.
Financial Metrics: Computed Weekly Returns, Risk-free rate adjustments, and Sharpe Ratios.
Data Visualization: Created scatter plots and trend lines to visualize the security characteristic line (SCL).
Key Findings
NVDA Beta: ~2.96 (High sensitivity to market movements).
Market Correlation: Approximately 25% of NVDA's variance is explained by the S&P 500 index.
